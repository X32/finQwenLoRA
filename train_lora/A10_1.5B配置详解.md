## 🔧 A10训练1.5B的三大配置方案

### 方案1: 极限性能 (推荐追求最高效果)

```bash
#!/bin/bash
# A10 - Qwen-1.5B 极限性能配置

MODEL="Qwen2.5-1.5B-Instruct"
BATCH_SIZE=16                   # 超大batch
LORA_RANK=256                   # 极限rank (vs 1080Ti的16)
LORA_ALPHA=512                  # alpha = 2*rank
TARGET_MODULES="全部线性层"     # 完整覆盖
NUM_EPOCHS=5                    # 更多训练轮次
LEARNING_RATE=8e-5

# 预期性能
显存使用: 18GB / 24GB (75%)
训练时间: 2-3小时
准确率: 90-93%
训练参数: 60M (4%)
```
